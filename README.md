# Cene goriv - OPB_projekt
Pri projektni nalogi predmeta OPB bomo naredili aplikacijo, ki bo zajemala podatke o črpalkah, s poudarkom na cenah goriva. Uporabnik bo tako z uporabo aplikacije ugotovil, na kateri črpalki v svoji regiji lahko natoči najcenejše gorivo in kje je specifična vrsta goriva sploh na voljo.
